Este projeto utiliza ASP.NET Core Minimal API com Entity Framework Core e MySQL para persistência de dados. Implementa autenticação JWT Bearer para segurança e Swagger/OpenAPI para documentação automática dos endpoints.
A arquitetura segue o padrão de Clean Architecture com separação em camadas de Domínio, Infraestrutura e API,
suportando operações CRUD para Administradores e Veículos com autorização baseada em roles.
